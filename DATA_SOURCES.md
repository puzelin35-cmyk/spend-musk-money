# 数据口径

本页面是娱乐模拟，净值、汇率、彩票奖金和工资都会变化。为保持游戏结果稳定，页面采用以下固定口径。

| 指标 | 页面取值 | 来源/说明 |
| --- | ---: | --- |
| 马斯克财富 | 9971 亿美元 | Forbes Real-Time Billionaires，2026-07-03 附近口径 |
| 美元兑人民币 | 6.78015 | 2026-07-03 附近公开汇率数据 |
| 折合人民币预算 | 6,760,487,565,000 元 | 997,100,000,000 × 6.78015 |
| 彩票大奖 | 5,000,000 元/天 | 娱乐口径：每天中一次 500 万 |
| 平均工资 | 71,590 元/年 | 国家统计局 2025 年城镇私营单位就业人员年平均工资 |

由此计算：

- 每天中 500 万，需要约 1,352,098 天，约 3,704 年。
- 按 71,590 元年均工资，需要约 94,433,406 年。
